üöÄ How to Connect a Client

To expose a local service through this tunnel server, run the following command on your local machine:

Basic usage:

go run github.com/matst80/showoff/cmd/client \ --host {{.ServerHost}} \ --name myapp \ --token {{.Token}} \ --target 127.0.0.1:3000

Example with local web server on port 5173:

go run github.com/matst80/showoff/cmd/client \ --host {{.ServerHost}} \ --name frontend \ --token {{.Token}} \ --target 127.0.0.1:5173

Parameters:

  • --host: This server's hostname
  • --name: Unique name for your tunnel (used in subdomain)
  • --token: Authentication token
  • --target: Your local service address

Access your service: Once connected, your service will be available at http://yourname.{{.BaseDomain}} or http://{{.ServerHost}}/yourname/
